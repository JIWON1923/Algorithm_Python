## Greedy
#### - 현재 상황에서 지금 당장 좋은 것만 고르는 방법
#### - 나중에 미칠 영향은 고려하지 않는다.
#### - 창의력, 최소한의 아이디어 능력을 검증하기 위해 출제되는 문제
#### - '가장 큰 순서', '가장 작은 순서' 등의 기준이 제시된다.
- 거스름돈 ( change.py )
- 큰 수의 법칙 ( law_of_large_number.py )
